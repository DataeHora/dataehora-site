# Assets Directory

This directory is reserved for static media files used across the site.

## Recommended Organization

- **images/** - Site images, logos, icons
- **fonts/** - Custom web fonts (if any)
- **media/** - Videos, audio files, or other media content

## Usage

Reference assets from HTML using the `/assets/` path:
```html
<img src="/assets/images/logo.png" alt="Site Logo">
```

## Notes

- Keep file sizes optimized for web performance
- Use appropriate image formats (WebP, PNG, JPG, SVG)
- Consider using CDN for large media files
